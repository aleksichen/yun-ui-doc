# 文字排版 Typography

YunUI 定义了一套基础的文本排版规则。

### 代码示例

h1-h6

<div class="yun-h1">h1</div>
<div class="yun-h2">h2</div>
<div class="yun-h3">h3</div>
<div class="yun-h4">h4</div>
<div class="yun-h5">h5</div>
<div class="yun-h6">h6</div>

``` css
<div class="yun-h1">h1</div>
<div class="yun-h2">h2</div>
<div class="yun-h3">h3</div>
<div class="yun-h4">h4</div>
<div class="yun-h5">h5</div>
<div class="yun-h6">h6</div>
```

small

<small>small</small>

``` css
<small class="yun-small">small</small>
```
